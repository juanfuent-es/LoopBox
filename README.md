# LoopBox
Compositor visual de oscilaciones 2D con Tone.js y p5.js.

## Uso rápido
1. Ejecuta `npm install` en `/home/runner/work/LoopBox/LoopBox`.
2. Inicia un servidor estático, por ejemplo: `python3 -m http.server 4173`.
3. Abre `http://127.0.0.1:4173/index.html` en un navegador moderno.
4. Pulsa **Iniciar audio** para activar Tone.js.
5. Añade/elimina layers y ajusta frecuencia, amplitud, velocidad y fase.
6. Arrastra en el canvas para desplazar fase/amplitud del layer activo y generar interferencias.

La línea blanca representa la suma de todas las capas oscilantes.
